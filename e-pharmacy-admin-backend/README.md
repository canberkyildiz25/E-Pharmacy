# E-Pharmacy Backend

## Setup
1. npm install
2. Configure .env with your MongoDB URI
3. npm run seed (to seed initial data)
4. npm run dev (development) or npm start (production)

## API Endpoints
- POST /api/user/login
- GET /api/user/logout
- GET /api/user/user-info
- GET /api/dashboard
- GET /api/orders
- GET /api/products
- POST /api/products
- PUT /api/products/:id
- DELETE /api/products/:id
- GET /api/suppliers
- POST /api/suppliers
- PUT /api/suppliers/:id
- GET /api/customers
- GET /api/customers/:id
